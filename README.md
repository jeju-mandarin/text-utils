# ⌨️ 실시간 글자 수 / 바이트 세기

텍스트를 입력하면 해당 텍스트의 통계를 실시간으로 분석해주는 간단한 웹 유틸리티 도구입니다.

🔗 **[지금 바로 사용해보기 (Live Demo)](https://jeju-mandarin.github.io/text-utils)**
*(위 링크의 `text-utils` 부분은 실제 생성하신 저장소 이름으로 변경해주세요.)*

---

## ✨ 주요 기능

* **실시간 분석:** 텍스트를 입력하거나 붙여넣는 즉시 모든 통계가 업데이트됩니다.
* **다양한 측정 항목:**
    * 글자 수 (공백 포함)
    * 글자 수 (공백 제외)
    * Byte 수 (UTF-8 기준)
    * 단어 수
    * 줄 수
* **시간 측정 기능:**
    * **예상 읽기 시간:** 평균 읽기 속도(약 250 WPM)를 기준으로 예상 시간을 계산합니다.
    * **입력 경과 시간:** 첫 글자를 입력한 순간부터 실제 경과 시간을 측정합니다.
* **타이머 컨트롤:**
    * **일시정지 / 계속:** 경과 시간 타이머를 멈추거나 이어서 계속할 수 있습니다.
    * **새로시작:** 경과 시간 타이머만 0초로 리셋하고 다시 시작합니다.
* **전체 초기화:** 작성한 텍스트와 타이머 등 모든 내용을 한 번에 초기화합니다.
